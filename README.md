# ⌨️ Keyboard Key Checker

A simple JavaScript project that displays information about any key you press on your keyboard in real time.

---

## 🚀 Features

- Detects any key press instantly
- Shows:
  - Key name (`e.key`)
  - Key code (`e.keyCode`)
  - Physical key (`e.code`)
- Clean UI with dark theme
- Smooth display update on every key press

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)
